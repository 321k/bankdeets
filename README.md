# <BankDeets/>
A react component for collecting bank details in any country and currency.

# Example
bankdeets.co

# API
* submitURL="where to post the form data to"
* Example: "<BankDeets submitURL="bankdeets.co/bankdeets" />"

* Swedish
* Norwegian
* Danish
* Finnish
* French
